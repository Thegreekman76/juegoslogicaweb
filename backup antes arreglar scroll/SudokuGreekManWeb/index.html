🧩 Guía Completa de Sudoku

🎯 Reglas Básicas

El objetivo es completar un grid de 9×9 con dígitos del 1 al 9, siguiendo estas reglas:

  • Cada fila debe contener los números 1-9 sin repetir
  • Cada columna debe contener los números 1-9 sin repetir
  • Cada caja 3×3 debe contener los números 1-9 sin repetir

🎮 Controles

  • Seleccionar celda: Haz clic en cualquier celda vacía
  • Ingresar número: Usa los botones numerados del 1-9
  • Modo notas: Activa "📝 Notas" para marcar candidatos
  • Borrar: Usa "🗑️ Borrar" para limpiar una celda
  • Pistas: Usa "💡 Pista" cuando estés atascado (limitadas)

🎨 Indicadores Visuales

  • Azul claro: Números dados (no modificables)
  • Dorado: Celda seleccionada
  • Azul celeste: Números iguales al seleccionado
  • Amarillo: Fila, columna y caja relacionadas
  • Rojo: Error detectado

💡 Estrategias Básicas

  • Eliminación: Si 8 casillas de una fila tienen números, la novena debe ser el faltante
  • Único candidato: Si una celda solo puede tener un número, ese es el correcto
  • Lugar único: Si un número solo puede ir en una celda de una fila/columna/caja
  • Usa notas: Marca posibles números para encontrar patrones

🏆 Niveles de Dificultad

  • 🟢 Fácil: 45-50 números dados, técnicas básicas
  • 🟡 Medio: 35-40 números dados, requiere más análisis
  • 🔴 Difícil: 28-32 números dados, técnicas avanzadas
  • ⚫ Experto: 22-26 números dados, máximo desafío
